MongoDB

Qué Es?

  • Base de Datos Open Source
  • Orientada a Documentos (Objetos)
  • Performance, Escalabilidad y Disponibilidad
  • No se necesita ORM

Ducuments

  • Registro (tupla, fila)
  • Estructura de datos (pares clave-valor)
  • "JSON"
  • Pueden tener otros docuemtosn, arrays o arrays de documentos

Collections

  • Almacenamiento para documentos
  • Analogas a las tables
  • No restringe esquemas
  • Clave primaria (_id)

_id se asigna por default

Instalación

  • Linux
  • macOS
  • Windows

Shell

  • $ mongo o $ mongo.exe
  • use [db] seleccionar db
  • db.[collection].insert(document) insertar documento
  • db.[collection].find(query) busca documentos
  • db.[collection].update(query, updates, ...) modifica documentos
  • db.[collection].remove(query) borra documentos
  • db.[collection].drop() vacia una coleccion
