Arquitectura

Event Driven

Paradigma de programacion donde el flujo de un programa es determinado por eventos como acciones del usuario, salida de sensores o mensajes de otros programas/procesos.

Event Loop

Es el corazon de Node.js, responsable de registrar las operaciones asincronicas.

Mecanismo principal que escucha eventos y ejecuta callbacks.

EventEmitter

Wrapper generico que nos permite crear API's orientadas a eventos.

Single Threaded

Corre en un solo thread basado en una tecnica de non-blocking I/O.

I/O se refiere a interaccion en disco y la red.

blocking

Ejecucion de codigo JS debe esperar hasta que el proceso NO JS termine.
